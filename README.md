# AsciiGen
Asciiart Generator\n
Sintaxe(python <FILE> <-Modo File/Cursed>)\n
O Modo File Cria um arquivo com "Ascii"+5 Numeros aleatorios+".txt"\n
O Modo Cursed Printa diretamente no console o seu Asciiart.\n
