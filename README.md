# AsciiGen
Asciiart Generator
Sintaxe(python <FILE> <-Modo File/Cursed>)
O Modo File Cria um arquivo com "Ascii"+5 Numeros aleatorios+".txt"
O Modo Cursed Printa diretamente no console o seu Asciiart.
